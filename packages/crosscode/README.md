# CrossCode
An OpenCode Remote Client CLI Tool.

```bash
npx crosscode
```

or, if using `pnpm`:

```bash
pnpm dlx crosscode
```

Scans the QR code with the CrossCode mobile app to connect to your local `opencode` server instantly.

> Requires `opencode` and `cloudflared` installed on your device.

## Features

+ No shared network required between devices.
+ No manual configuration of ports or IP addresses.
+ Just start the CLI and scan the QR on your mobile device to connect to your local `opencode` server.
